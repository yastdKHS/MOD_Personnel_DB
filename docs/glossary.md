# 用語集

本プロジェクトのドメイン固有の用語。実装・レビュー時の解釈のブレを防ぐための共通言語。

| 用語 | 説明 |
|---|---|
| 発令 | 人事上の決定（異動・補職・昇任等）を公式に発することの総称 |
| 辞令 | 発令の内容を個人に対して示す文書。本プロジェクトが扱うPDFの主な内容 |
| 補職 | ある官職・職務に就けること |
| 併任 | 本務を持ちながら別の職務も兼ねて任じられること |
| 兼務 | 併任と近い概念で使われる、複数の職務を同時に担うこと |
| 階級 | 自衛官の階級呼称（例: 陸将、1等陸尉 等）。時代によって呼称・区分が変わることがある |
| 官職 | 具体的な役職・ポスト |
| 異動 | 所属・職務の変更全般 |
| 発令日 | 発令の効力が生じる日。PDFの公表日と一致しない場合がある点に注意 |
| 様式 / レイアウト | 発令PDFの版面構成（列構成・見出し表現等）。時期によって複数の様式が存在する（`layouts/` 参照） |
| 来歴（Provenance） | あるデータがどの原資料・どの処理過程を経て生成されたかの追跡情報 |
| ゴールデンファイル | あるサンプル入力に対する「正解」として管理される期待出力（`sample_outputs/` 参照） |
| 名寄せ | 表記ゆれ・異体字等を持つ同一人物・同一組織のレコードを統合すること |

> このリストは初期案であり、実装を進める中で用語が増える。追加時は本ファイルを更新する。
